# 天问 · AskTheOracle — 项目总结

> 2026-07-03 · vA.0.8.5 发布

---

## 一、项目概述

天问是一个纯前端开源算卦应用，部署在 GitHub Pages，手机浏览器打开即用，支持 PWA 一键安装到桌面离线运行。

**技术栈**：HTML + CSS + Vanilla JS，零依赖，双击 `index.html` 即可本地运行。

**在线地址**：https://luyi14-bits.github.io/AskTheOracle/

**许可证**：GNU AGPL v3.0

---

## 二、当前功能（八模式就绪）

| 模式 | 说明 |
|------|------|
| 🪙 铜钱掷爻 | 三枚铜钱掷六次，生成本卦 + 变卦，朱熹变占法解读 |
| 🌸 梅花易数 | 三数起卦，互卦推演，体用生克定吉凶 |
| 📝 姓名起卦 | 姓名笔画数起卦，繁体康熙字典笔画标准，终身卦解读 |
| 🎂 生辰起卦 | 公历/农历双输入，年月日时法起终身卦 |
| 🔮 掷筊问杯 | 传统筊杯掷六次，圣杯/笑杯/阴杯判定 |
| 🖐 掐指小六壬 | 月日时三推掌诀，大安/留连/速喜/赤口/小吉/空亡六神断吉凶 |
| 🔗 交叉起卦 | 姓名 + 生辰八字合参，看名字与命格是否契合 |
| 📊 五格姓名学 | 天格/人格/地格/总格/外格 81 数理 + 三才五行配置 |

### 其他特性

- **白话解读** — 64 卦 × 100 字白话总结，普通人也能看懂卦象
- **分享卡片** — Canvas 生成卦象分享图，一键下载
- **PWA 离线** — Service Worker 缓存，手机安装到桌面断网可用
- **动画音效** — 铜钱抛掷、筊杯落地等流畅动画 + 音效反馈
- **水印保护** — "仅供娱乐 · luyi14-bits 全栈开源项目" 全页标注

## 四、技术亮点

1. **纯前端零依赖** — 不需要 Node.js、npm、CDN，单 HTML 文件即可运行
2. **完整的易经六十四卦数据** — 卦符、卦辞、爻辞、白话解读全部内置
3. **朱熹变占法** — 铜钱模式遵循传统动爻变卦规则
4. **梅花易数体用生克** — 互卦推演、体用五行判定
5. **农历转换** — 公历→农历纯 JavaScript 算法，不依赖后端
6. **康熙字典笔画标准** — 3000+ 汉字笔画库，姓名起卦/五格数理共享
7. **Canvas 分享卡片** — 离线生成卦象分享图，含水印
8. **PWA 全栈** — manifest + Service Worker + 独立图标
9. **CSP 安全策略** — Content-Security-Policy 不包含 `unsafe-inline`
