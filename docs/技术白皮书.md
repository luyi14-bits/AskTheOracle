## 一、项目概述

### 1.1 定位

「天问」是一个零依赖、可离线、可安装的纯前端占卜应用，将中国传统数术（周易六十四卦、梅花易数、小六壬、姓名学等）转化为现代化的交互工具。用户无需联网、无需安装任何软件，双击 `index.html` 即可使用，也可通过 PWA 安装到手机桌面。

### 1.2 核心理念

- **离线优先**：Service Worker 全量缓存，断网如常
- **零依赖**：不引入任何 CDN/framework/npm 包，纯手写原生 JS
- **数据驱动**：64 卦 384 爻完整经文、3000+康熙字典笔画、200 年农历天文数据全部内嵌
- **渐进增强**：从最基础的铜钱掷爻到梅花易数、姓名起卦、小六壬、圣杯问筊，五种算法覆盖八成以上华人占卜场景

## 三、7 种起卦算法详解

### 3.1 铜钱掷爻 (v0.2)

**算法**: `castOnce()` 模拟三枚铜钱随机抛掷，每枚铜钱 50% 正面/反面。三枚组合决定爻的性质：

| 正面数 | 结果 | 映射 |
|--------|------|------|
| 2 或 0 | 阳爻 (━━━) | 50% |
| 1 或 3 | 阴爻 (╌ ╌) | 50% |
| 3 或 0 | 动爻 (⚡) | 25% |

6 次掷爻完形 → `findGua(yaoLines)` 查 `GUA_BY_BINARY` 表 → `getBianGua()` 计算变卦 → `getInterpretationFocus()` 朱熹变占法规则判定。

**动画**: `AnimateCoinFlip()` — `@keyframes coinFlip`: `rotateY(0→720deg)` + `scale(0→1.2→1)`，
