## 老板的点子：姓名 + 生辰交叉起卦 ✅ 可行

### 怎么交叉？

把现有两种起卦模式的**数字来源混合**，出一卦：

```
上卦数 = 姓氏笔画 + 出生年地支数（子1~亥12）
下卦数 = 名字总笔画 + 出生月份
动爻数 = 出生日期 + 出生时辰（子1~亥12）
```

取模后进入现有六十四卦解读流程。用户只填一次表单（名字 + 生日），背后自动交叉计算。

### 价值

- 单一「姓名起卦」只反映名字本身的运势
- 单一「生辰起卦」只反映命格本身的走势
- **交叉起卦**看的是"这个名字配你这个命，合不合"——更贴近普通人想问的问题

### 开发量

复用现有 `stroke-data.js`（已有 3000+ 字笔画库）+ 现有梅花易数起卦逻辑，新增 ~100 行。

## 同期推荐：另外两个高价值方向

### 🥈 小六壬 — "手掌上的算卦"

中国传统民间占卜，只用左手掌 6 个位置（大安/留连/速喜/赤口/小吉/空亡），月→日→时推三步出结果。

| 优势 | 说明 |
|------|------|
| 超简单 | 6 个结果，不需要理解卦象 |
| UI 好看 | 手掌动画，拇指依次点过六个位置 |
| 适合传播 | 比六十四卦轻量，社交分享更友好 |
| 代码量小 | ~80 行 JS + ~50 行 CSS |

### 🥉 五格姓名学 — "名字能打多少分"

利用现有 3000+ 字笔画库，计算天格/人格/地格/总格/外格五格数理，查 81 数理吉凶表。

叠加到现有姓名起卦页面，卦象解读下方展示五格评分 + 三才配置，形成完整「姓名评估报告」。
