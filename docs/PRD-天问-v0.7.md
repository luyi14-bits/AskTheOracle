## 1. Executive Summary

**问卦 v0.7** 新增第六种起卦模式"掷筊问杯"（筊杯/圣杯），并为所有六十四卦补充白话解读。用户不再只能掷铜钱和数字，还能像在庙里一样双手合握筊杯掷地问天；也不再面对看不懂的文言卦辞，每卦都有一句人话告诉你"这对你的生活意味着什么"。

---

## 2. Problem Statement

### 用户痛点

**圣杯侧**：
- 很多用户（尤其是港台/闽南/东南亚华人）从小在庙里掷筊杯，对这种占卜方式有天然的熟悉和亲近
- 当前问卦有铜钱、三数、时间、生辰、姓名五种方式，但缺少最民间、最接地气的"掷筊问杯"

**白话侧**：
- 文言卦辞像"元亨利贞""利牝马之贞"，普通用户看完不知道该怎么办
- 同赛道产品大多只给文言原文，"算完还是不懂"是用户最高频的抱怨

### 业务影响

- 掷筊是极具文化辨识度的功能，能显著差异化于同赛道工具
- 白话解卦让产品从"易经词典"变成"算卦顾问"，解决"看不懂"的核心痛点

## 4. User Personas

### Persona F（新增）：庙里常客 — 阿达

- **年龄**: 30-60 岁，港台/闽南/东南亚文化背景
- **场景**: 从小跟着长辈去庙里掷筊杯，现在想在网上掷杯
- **行为**: 点"掷筊问杯"→看到筊杯动画→圣杯！开心→笑杯…再来→六掷出卦→读白话解读
- **需求**: 仪式感要到位（筊杯要有动画），结果要好懂（不需要知道"元亨利贞"是什么）

### Persona G（新增）：易经小白 — 小雅

- **年龄**: 20-30 岁，完全不懂易经
- **场景**: 好奇心驱使，想算一卦但怕看不懂
- **行为**: 任一模式起卦→看结果→直接拉到"💬 白话解读"→读懂→截图
- **需求**: 不看文言文也能理解卦象含义

## 5. User Stories

### US-1: 掷筊问杯起卦（P0）

```
Feature: 圣杯掷筊六次起卦

  Scenario: 用户成功掷筊获得完整卦象
    Given 用户点击"掷筊问杯"标签
    When 点击"掷杯"按钮
    Then 弹出"静心凝神，心中默念所求"提示
    And 三秒后筊杯动画从天而降
    And 显示本次结果（圣杯/笑杯/阴杯）
    And 自动进入下一掷（共六次）
    And 六掷完成展示本卦+变卦+完整解读

  Scenario: 掷筊六爻映射
    Given 用户掷出圣杯（一正一反，~50%概率）
    Then 记录为阳爻(⚊)，不变
    Given 用户掷出笑杯（双阳面朝上，~25%概率）
    Then 记录为阴爻(⚋)，不变
    Given 用户掷出阴杯（双弧面朝上，~25%概率）
    Then 记录为阳爻(⚊)，但标记为"会变动"→可变阴

  Acceptance Criteria:
  - [ ] 六次掷筊串行，间隔 3s（含动画时间）
  - [ ] 三种结果概率近似 50% / 25% / 25%
  - [ ] 本卦+变卦通过现有 GUA_BY_BINARY 引擎正确查表
  - [ ] 掷筊期间标签切换被禁用
```

### US-2: 白话解卦（P0）

```
Feature: 每卦附带白话解读

  Scenario: 任何模式起卦后都有白话
    Given 用户在任意模式（铜钱/三数/时间/生辰/姓名/圣杯）下获得卦象
    When 结果区渲染完成
    Then 在卦辞下方出现"💬 白话解读"区块
    And 区块内展示 60-120 字白话文
    And 变卦（如有）同样展示白话解读

  Scenario: 梅花易数体用生克也有白话
    Given 梅花/生辰/姓名模式下有体用生克卡片
    When 结果渲染
    Then 生克结论下方加一句白话总结（如"用生体，大吉——当前形势对你有利，顺势而为"）

  Acceptance Criteria:
  - [ ] 64 卦每卦 `plain` 字段非空
  - [ ] plain 长度 60-120 字之间
  - [ ] 白话区块在视觉上区别于文言卦辞（浅背景色）
```

## 8. Technical Considerations

### 8.1 圣杯→爻映射（修订版）

原方案只有阴杯能发动变爻，导致阴阳爻发动不对称。修订后：

| 掷筊结果 | 概率 | 映射爻型 | changing |
|----------|:---:|----------|:--------:|
| 圣杯（一正一反） | ~50% | 阳爻 | `false` |
| 笑杯（双正面朝上） | ~25% | 阴爻 | `false` |
| 阴杯（双弧面朝上） | ~25% | 阳爻 | `true` (老阳) |

**设计理由**：筊杯只有三种物理状态，无法完全对应四爻型。选择让"阴杯"对应老阳（阳爻会变阴），保证阴阳都能发动变卦。折中在解读文案中说明"掷筊问杯的变卦仅代表部分天机变动"。

### 8.2 筊杯动画

```javascript
// animation.js 新增
animateJiaobei: function(container, result) {
    // 1. 创建两只弯月形 div (.jiaobei-piece)
    // 2. CSS animation: translateY(从天而降) + rotate(翻转) + opacity
    // 3. 落地后展示正反面: 一正一反/双正/双反
    // 4. 圣杯→绿色辉光, 笑杯→黄色辉光, 阴杯→红色辉光
}
```
