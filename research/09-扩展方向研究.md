# 天问扩展方向研究 · 综合报告

> 调研日期：2026-07-03
> 当前状态：A 0.7 已发布，六模式就绪（铜钱/梅花/姓名/生辰/筊杯/PWA）
> 调研范围：姓名+生辰交叉 / 五格剖象 / 小六壬 / 灵签 / 每日运势

---

## 方向一：姓名 + 生辰八字交叉起卦

### 方案 A：姓名笔画 + 生辰数字 双因子起卦

**原理**：梅花易数允许多源数字组合。姓名给出三数（姓氏笔画、名字1笔画、名字2笔画），生辰给出三数（年支、月、日），两者组合得六数，分别映射上卦/下卦/动爻/互卦/变爻，形成更丰富的终身卦。

**规则示例**：
1. 姓氏笔画 → 上卦基数
2. 出生年地支数（子1…亥12）→ 上卦偏移
3. 名字总笔画 → 下卦基数
4. 出生月份 → 下卦偏移
5. 出生日期 → 动爻基数
6. 出生时辰（子1…亥12）→ 动爻偏移

上卦 = (姓氏笔画 + 年支数) % 8，下卦 = (名字总笔画 + 月份) % 8，动爻 = (日期 + 时辰) % 6

**解读优势**：
- 输出「本卦 + 互卦 + 变卦 + 姓名五行 + 生辰五行」五层信息
- 做一个交叉评分：姓名卦 × 生辰卦 的体用生克关系，看名字是否补益命格

### 方案 B：八字五行 + 姓名五行 生克匹配度

**原理**：将生辰八字日主五行（金木水火土）与姓名五格数理所对应的五行做对比，得出名字与命格的匹配度评分。

**实现难度**：中等。需要：
- 八字排盘 → 日主五行
- 五格数理 → 人格五行
- 两者比对：相生则吉、比和则中、相克则凶

## 方向二：五格剖象姓名学

### 原理

五格剖象法是日本明治时期熊崎健翁创立的姓名学体系，传入中国后广泛流传。

**五格**：
| 格 | 计算 | 含义 |
|------|------|------|
| 天格 | 姓氏笔画 + 1（单姓） | 祖运、先天 |
| 人格 | 姓氏笔画 + 名字首字笔画 | 主运、性格（核心） |
| 地格 | 名字全部笔画（单名+1） | 前运、38 岁前 |
| 总格 | 姓名全部笔画 | 后运、38 岁后 |
| 外格 | 总格 - 人格 + 1 | 社交运、副运 |

**81 数理吉凶**：每个格的笔画数对应 1-81 的数理吉凶表：
- 大吉（1/3/5/6/7/8/11/13/15/16…）
- 凶（2/4/9/10/12/14/19/20/22/26/27/28…）
- 半吉（其他）

**三才配置**：天格→人格→地格的五行相生相克关系

### 对我们项目的价值

天问已有姓名起卦（笔画→卦象），叠加五格分析能形成一个完整的「姓名评估报告」：
- 上半部分：姓名起卦（卦象 + 体用生克）
- 下半部分：五格数理（天/人/地/总/外 五格吉凶）
- 底部总结：综合评分

## 方向三：小六壬（掌诀占卜）

### 原理

小六壬是中国传统民间占卜法，仅用左手掌六个位置：

| 掌诀 | 含义 |
|------|------|
| 大安 | 平安、顺利、大吉 |
| 留连 | 拖延、犹豫、中平 |
| 速喜 | 喜事将至、快速、小吉 |
| 赤口 | 口舌是非、小凶 |
| 小吉 | 小有收获、中吉 |
| 空亡 | 事不长久、大凶 |

**推算**：月→日→时三点，每次从起算位顺时针数到目标数即可。

### 优势

- 超简单：不需要生成完整卦象，6 个结果一目了然
- 适合"今日运势""出门问事"等快问场景
- UI 精美：可以做手掌动画，拇指依次点过六个位置
- 代码量极小：~50 行 JavaScript

## 方向四：灵签抽签

### 原理

传统庙宇 100 签（或 60 签/64 签），求签者摇签筒后抽一支签，根据签号查看签文。

### 对我们项目的适配

用 64 卦代替 100 签：让用户"摇签筒"（动画），随机出一卦，解读方式和现有六十四卦系统完全复用。

只需要加：
- 摇签动画
- 独立 Tab

---

## 方向五：每日一卦 / 每日运势

### 原理

以公历日期为种子，每天为所有用户展示同一个卦。类似"今日运势"卡片。

### 实现

- `seed = YYYYMMDD % 64`，取对应卦
- 每天早上自动更新
- 可做成 PWA 小组件/快捷入口

---

## 方向六：姓名+生辰交叉起卦（重点）

### 合并总结

这个方向可以拆成三层实现：

| 层 | 内容 | 难度 | 用户体验 |
|------|------|------|---------|
| **L1 - 基础** | 姓名笔画+生辰数字 双因子起一卦 | 低 | 输入名字+生日 → 1 个终身卦 |
| **L2 - 分析** | L1 + 五行对比（名字五行 vs 八字日主） | 中 | 多了五行匹配度评分 |
| **L3 - 全面** | L2 + 五格剖象 + 三才配置 | 中高 | 完整姓名命理报告 |

建议从 L1 起步，复用现有姓名起卦代码 + 现有梅花易数生辰起卦逻辑，最小改动交付。

## 综合推荐优先级

| 排名 | 方向 | 理由 | 估时 |
|------|------|------|------|
| 🥇 | 姓名+生辰 L1 交叉 | 老板点名，现有代码复用率高 | 0.5 天 |
| 🥈 | 小六壬 | 超简单、差异化大、仪式感强 | 0.5 天 |
| 🥉 | 五格姓名学 | 丰富姓名模式，数据可复用笔画库 | 0.5 天 |
| 4 | 每日一卦 | 极简，提升用户留存 | 2h |
| 5 | 灵签抽签 | 仪式感强，代码复用 64 卦 | 3h |

---

## 结论

天问已覆盖六种起卦模式，下一步最优策略是：
1. **做深**而非做宽 — 姓名+生辰交叉起卦是用户明确需求
2. 同时做一个小而美的**小六壬**，因为 UI 差异化大、容易被传播
3. 长期可布局**每日运势**提升日活

8 个方向全部完成后天问将从"多模式算卦工具"升维为「易学百宝箱」。
